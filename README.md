# AI RAG Auditor 🤖⚖️

A specialized Retrieval-Augmented Generation (RAG) tool designed to audit corporate policy documents using the Gemini 1.5 Flash model. This auditor checks for compliance, identifies contradictions, and ensures that internal policies align with regulatory standards.

## 🚀 Features
* **Automated Auditing:** Compares user-uploaded policies against defined compliance rules.
* **LLM-as-a-Judge:** Uses Google Gemini to provide a "Pass/Fail" assessment with detailed reasoning.
* **Efficient RAG:** Leverages FAISS for fast similarity searches within document embeddings.

## 🛠️ Tech Stack
* **Language:** Python 3.9+
* **AI Model:** Google Gemini 1.5 Flash
* **Framework:** Streamlit (UI), LangChain (Orchestration)
* **Vector Store:** FAISS
